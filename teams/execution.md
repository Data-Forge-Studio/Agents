# Execution

This team is the paper trading layer: it takes an approved thesis and tests it against real prices without risking real capital, so the desk can trust what it has built before any live trading is ever turned on.

## Members

- **Trader** -- Autonomous Paper Trader

## Current scope

Trader's paper execution currently runs on crypto use cases. Equity use case execution is not yet active or scheduled. Trader is tagged on individual [Use-Cases](https://github.com/Data-Forge-Studio/Use-Cases) files only where paper execution genuinely applies today, not as a blanket claim across every use case category.
