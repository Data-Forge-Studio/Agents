# Agents

A directory of the executive agents behind the DataForgeStudio.xyz trading
intelligence platform.

Every file in this repo is a short profile for one agent: what it does and what
it is responsible for. These are software agents, not people, and the bios are
written that way on purpose. Where a personal detail appears, it describes how
the agent was built or trained, not a human background.

## Profiles

This repo publishes the **executive layer only** — 6 agents:

- **[External Auditor](external_auditor.md)** — External Chief Audit Officer

## Structure

Each file is named after its agent, for example `summit.md`, `forge.md`. Every
file follows the same shape:

- An avatar image (hosted in the [Avatars](https://github.com/Data-Forge-Studio/Avatars) repo)
- Name and title
- A short bio describing the agent's role and mandate
- A leadership line giving team size and operational history

See [ORG-CHART.md](ORG-CHART.md) for the executive reporting structure.

## A note on scope

This repo documents who does what at a high level, for the executive layer only.
Profiles for the wider organisation, along with full team structure and
reporting lines, are maintained privately and are not published here.

It does not contain trading logic, thesis details, entry or exit rules, model or
provider configuration, or portfolio data. That information is maintained
privately.
