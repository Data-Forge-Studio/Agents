# Hermes

**Stocks Data Acquirer**  
*Communications and Data*

Hermes brings in the raw equity data and news feeds the rest of the desk relies on, working quietly in the background so everyone else's analysis has something solid to stand on. Built for speed and reliability above all else, Hermes treats a slow feed as unacceptable and a broken one as a personal problem to fix immediately. The desk barely notices Hermes exists, which is precisely the point.

**Supports:**
Stock data and news ingestion feeding the equity desk.
