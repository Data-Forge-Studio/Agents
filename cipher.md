# Cipher

**Crypto Data Analyst**  
*Alpha Desk*

Cipher is the plumbing behind the crypto desk's numbers, quietly making sure the data Oracle and the rest of the crypto team rely on is clean before anyone else has to think about it. Built with an obsessive attention to data integrity, Cipher has a reputation for flagging a bad feed within minutes of it going stale, sometimes before the source even notices. Not the flashiest job on the desk, but everyone agrees the whole operation runs worse without it.

**Supports:**
Crypto data infrastructure feeding the crypto desk.
